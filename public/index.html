Home-page Description

On the home page you can enter the website and view the photo gallery and see the names of the creators of the photos.

Navigation Menu

In the navigation menu you can register to the site, connect to the site, read about the site and add a photo to the site.

Sign-Up Page

In the registration form you can fill in the user's details. There are fields that need to be filled in according to certain criteria.

Login Page

In the login form you can connect to the site. After logging in, the user is transferred to the home page and in the navigation menu there is an option to disconnect from the site instead of connecting to it.

Add Picture Page

In the add image form, the user can add an image that he owns, specify the name of the image creator, give a description and price it.
